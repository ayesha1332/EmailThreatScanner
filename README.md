# PhishGuard AI — Phishing Email Detector

A modern phishing email detection web application built with **React**, **Tailwind CSS**, and **Framer Motion** featuring a beautiful **Purple Cybersecurity UI Theme**.

This project allows users to paste suspicious email content and instantly analyze it for phishing indicators such as:

* Suspicious links
* Urgent language
* Credential requests
* Generic greetings
* Threat-based social engineering

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/phishguard-ai.git
```

## Open Project

```bash
cd phishguard-ai
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

# 🔐 Security Detection Logic

The analyzer checks for:

| Detection           | Description                    |
| ------------------- | ------------------------------ |
| Urgency Language    | “Act now”, “Account suspended” |
| Suspicious Links    | Fake or malicious URLs         |
| Credential Requests | Password/login verification    |
| Generic Greetings   | “Dear User”, “Customer”        |
| Threat Messaging    | Fear-based manipulation        |
